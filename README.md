## Descripción

Este proyecto consiste en la implementación de un **Árbol AVL** en el lenguaje de programación C#.
Un árbol AVL es un tipo de árbol binario de búsqueda que se mantiene balanceado automáticamente para mejorar el rendimiento en operaciones como inserción y búsqueda.

El objetivo de este programa es mostrar cómo funciona este tipo de estructura de datos, incluyendo las rotaciones necesarias para mantener el equilibrio del árbol.

## Funcionalidades del programa

Inserción de nodos
Balanceo automático del árbol
Rotaciones (izquierda y derecha)
Recorrido en orden (muestra los datos ordenados)


## Estructura del proyecto

El proyecto está dividido en 3 clases principales:

* `NodoAVL.cs` → Representa cada nodo del árbol
* `ArbolAVL.cs` → Contiene la lógica del árbol (insertar, balancear, rotaciones)
* `Program.cs` → Ejecuta el programa principal
